- public: 公共文件，包含项目的入口 HTML 文件和图标。
- src：源代码目录。
- App.js: 根组件
- index.css: 全局 CSS 样式文件
- index.js: 应用的入口 JS 文件。
- package.json: 项目的配置文件，定义项目的依赖关系、安装插件的版本
- api：后端接口
- assets：图片等静态资源
- pages：存放 jsx 文件，也就是页面
- router：路由配置
- store: 状态管理
- utils: 工具类
